### Create Environment

```powershell
python -m venv myenv
```

### Activate Environment

```.cmd
myenv\Scripts\activate
```

### Deactivate Environment

```.cmd
deactivate
```

### Install Requirements

```powershell
pip install -r requirements.txt
```

### Install CLI Global

```powershell
pip install .
```

### Validate Installation

```powershell
pip show sf-users-cli
```

### Install Inside Env

```powershell
pip install -e .
```
